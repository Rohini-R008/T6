# Laundry Mart Login Page

## Description
This project implements a login page for a Laundry Mart web application using HTML and CSS Flexbox. The layout contains an image section on the left and a login form on the right.

## Features
- Flexbox layout
- Centered login form
- Email and password inputs
- Login button
- Register link

## Files
index.html
index.css

## Technologies Used
HTML
CSS (Flexbox)

## How to Run
Open index.html in any browser.
